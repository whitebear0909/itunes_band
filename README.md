## Command
### `yarn start`

Runs the app in the development mode.\
Open [http://localhost:3000](http://localhost:3000) to view it in the browser.

### `yarn test`

Launches the test runner in the interactive watch mode.

## About Project

### Project structure
    - assets
        neccessary assets
    - components
        reusable components
    - pages
        page components
    - services
        define global functions and constants.
    - App.css
        define styles
    - App.test.js
        unit test code here.
